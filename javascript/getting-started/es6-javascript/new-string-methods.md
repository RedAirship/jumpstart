# New String methods

ES6 gives us 4 new handy String methods:

`.startsWith()`

```javascript
"abc".startsWith("ab") // returns true
```

`.endsWith()`

```javascript
"abc".endsWith("bc") // returns true
```

* `.includes()`

```javascript
"abcde".endsWith("d") // returns true
```

* `.repeat()`

```javascript
"*".repeat(5) // returns "*****"
```
