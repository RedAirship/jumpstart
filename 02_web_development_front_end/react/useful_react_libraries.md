### Useful React libraries

- [react-motion](https://github.com/chenglou/react-motion) - For animating components
- [react-responsive](https://github.com/contra/react-responsive) - For making components mobile-responsive
- [react-strap](http://reactstrap.github.io/) - For integrating bootstrap UI framework
- [react-dotdotdot](https://github.com/CezaryDanielNowak/React-dotdotdot) - For truncating long components
